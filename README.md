# OrangeHRM Playwright Automation Framework

## Overview

This project demonstrates a Playwright + TypeScript automation framework using the Page Object Model (POM) design pattern.

## Tech Stack

* Playwright
* TypeScript
* Page Object Model (POM)
* GitHub
* CI/CD Ready

## Automated Scenarios

* Valid Login
* Invalid Login
* Logout
* Add Employee

## Framework Features

* Reusable Page Objects
* Test Data Management
* Centralized Constants
* Scalable Folder Structure
* Maintainable Test Design

## Execute Tests

```bash
npx playwright test
```

## Execute Tests in Headed Mode

```bash
npx playwright test --headed
```

## Generate Report

```bash
npx playwright show-report
```
